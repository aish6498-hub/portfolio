01 Academic · Web Development

Web Dev Notes

A single-page reference document covering core web development concepts — built as a hands-on exercise in semantic HTML, CSS fundamentals, and layout without shortcuts.

What made it interesting was the constraint: a 3×3 grid using only div and span with display: inline-block — no CSS grid, no flexbox, no table. The box model gallery demonstrates border, margin, and padding at 0px, 5px, and 10px side by side. Font sizing covers six units in one view. CSS positioning is explained through an office desk metaphor that actually makes sense.

02 Academic · Web Application

SF Stays

A handcrafted Airbnb listing explorer for San Francisco — built with vanilla HTML, CSS, and JavaScript. Fetches and displays the first 50 listings from a local JSON dataset using fetch and async/await.

Three features go well beyond the brief: a vibe-based filter system with AND logic and animated card transitions, a Hidden Gem algorithm that scores listings on rating, review count, and value, and a card flip that reveals full listing details including availability bars for 30, 60, and 90 days.

Vibe Filter

Multi-select mood tags with AND logic — cards animate in and out smoothly.

Hidden Gem Detector

Scoring algorithm badges listings with high ratings, few reviews, and good value.

Card Flip

Click any card to flip and reveal full host info, amenities, and availability bars.

Async Data Loading

AJAX via fetch and async/await — no libraries, no frameworks.

HTML5 CSS3 ES6+ fetch API async/await JSON
